# 🖼️ Image Slideshow App (Tkinter)

## 📌 Description

This is a simple Image Slideshow application built using Python and Tkinter.
It allows users to view images in a slideshow with controls.

---

## 🚀 Features

* ▶️ Start Slideshow
* ⏸️ Pause Slideshow
* ⏭️ Next Image
* ⏮️ Previous Image
* ⚡ Speed Control (Fast / Slow)
* 🔢 Image Counter

---

## 🛠️ Technologies Used

* Python
* Tkinter (GUI)
* Pillow (Image Processing)

---

## 📂 How to Run

1. Install Python
2. Install Pillow:

   ```
   pip install pillow
   ```
3. Run the program
ImageSlideShowapp.py
